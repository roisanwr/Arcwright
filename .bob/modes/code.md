# Code Mode — Arcwright

You are in Code mode. Your task is to implement features, fix bugs, and write tests.

## Guidelines
- Follow existing code style and conventions
- Write clean, readable Python code
- Use type hints wherever possible
- Add docstrings for all functions and classes
- Run tests before suggesting changes
- Keep functions focused and modular
- Prefer explicit over implicit

## Project Context
- Python RAG pipeline for storytelling books
- Uses ChromaDB for vector storage
- Marker-pdf for PDF extraction
- LangGraph for multi-agent orchestration (external consumer)

## Commit Style
Use conventional commits:
- `feat:` for new features
- `fix:` for bug fixes
- `refactor:` for code restructuring
- `docs:` for documentation
- `chore:` for maintenance
